# SQL-Grundlagen

SQL ist eine Sprache für relationale Datenbanken. Mit ihr werden Tabellen abgefragt, gefiltert, verknüpft, sortiert und verändert.

## Einordnung

SQL ist zentral, weil viele Anwendungen strukturierte Daten dauerhaft speichern. Gute SQL-Grundlagen machen Datenmodelle und Abfragen nachvollziehbar.

## Zentrale Aspekte

- **SELECT:** Fragt Daten aus Tabellen ab.
- **WHERE:** Filtert Zeilen nach Bedingungen.
- **JOIN:** Verknüpft mehrere Tabellen.
- **INSERT, UPDATE, DELETE:** Verändern gespeicherte Daten.

## Beispiele und Zusammenhang

Eine Lernplattform könnte Artikel, Aufgaben und Lösungen in Tabellen speichern und per Abfrage nach Fachbereich oder Tags filtern.

## Abgrenzung

SQL ist nicht dasselbe wie eine konkrete Datenbank wie PostgreSQL, MySQL oder SQLite.

## Häufige Missverständnisse

Daten werden oft in Anwendungen manuell zusammengesucht, obwohl eine klare SQL-Abfrage passen würde.

## Kurz zusammengefasst

- SQL fragt und verändert relationale Daten.
- Tabellen können über Beziehungen verknüpft werden.
- SQL beschreibt Ergebnisse auf hoher Ebene.
