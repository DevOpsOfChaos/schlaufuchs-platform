CSS bestimmt, wie HTML-Inhalte aussehen. Während HTML die Struktur einer Seite beschreibt, kümmert sich CSS um Farben, Abstände, Schriftgrößen und viele weitere Darstellungsfragen.

## Was ist CSS?

CSS steht für **Cascading Style Sheets**. Damit werden Regeln formuliert, die festlegen, wie bestimmte HTML-Elemente angezeigt werden sollen.

Eine typische CSS-Regel besteht aus:

- einem **Selektor**
- einer oder mehreren **Eigenschaften**
- passenden **Werten**

## Beispiel für eine Regel

Eine einfache Regel kann so aussehen:

`p { color: blue; }`

Das bedeutet: Alle Absätze sollen blau dargestellt werden.

## Warum HTML und CSS zusammengehören

HTML beschreibt den Inhalt und seine Struktur. CSS beschreibt die Darstellung. Erst zusammen entsteht daraus eine sauber aufgebaute Webseite, die auch visuell funktioniert.

Darum ist CSS kein Zusatzwissen, sondern ein direkter zweiter Grundpfeiler im Web-Einstieg.

## Typische erste CSS-Themen

Am Anfang ist es besonders wichtig, diese Dinge zu verstehen:

- Wie ist eine CSS-Regel aufgebaut?
- Was ist ein Selektor?
- Was sind Eigenschaften und Werte?
- Wie wirkt CSS auf HTML?

Wenn diese Basis sitzt, werden spätere Themen wie Layout, Box-Modell und responsive Gestaltung viel leichter.

## Merksatz

HTML gibt einer Webseite ihre Struktur, CSS gibt ihr Aussehen und sichtbare Form.
