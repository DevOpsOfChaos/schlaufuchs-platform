## Warum Flexbox?

Flexbox hilft dabei, Elemente **einfach anzuordnen**.

## Typischer Start

```css
.container {
  display: flex;
}
```

## Wichtige Eigenschaften

### justify-content

Steuert die Verteilung entlang der Hauptachse.

### align-items

Steuert die Ausrichtung auf der zweiten Achse.

### gap

Erzeugt Abstände zwischen Elementen.

## Wann ist Flexbox gut?

Vor allem für Navigationen, Button-Reihen und kleine Kartenbereiche.
