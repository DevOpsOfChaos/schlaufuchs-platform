## Was Selektoren eigentlich tun

CSS-Selektoren wählen Elemente aus. Erst danach kann ein Stil auf diese Elemente angewendet werden. Ohne Selektor gibt es also keine gezielte Gestaltung.

## Drei wichtige Grundtypen

- Elementselektoren treffen alle passenden HTML-Tags.
- Klassenselektoren treffen Elemente mit einer bestimmten Klasse.
- ID-Selektoren treffen genau ein Element mit dieser ID.
