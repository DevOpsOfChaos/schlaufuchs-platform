## Typische Fehlstelle

Die häufigste Fehlstelle ist: Grid wird für eine einfache flexible Reihe überladen. Dadurch wird ein kleiner fachlicher Unterschied unsichtbar. Besser ist, die Aufgabe erst auf die konkrete Prüffrage zu verkleinern und dann mit einer Gegenprobe abzusichern.
