CSS wirkt nicht einfach „auf die ganze Seite“, sondern gezielt auf bestimmte Elemente. Genau dafür braucht man **Selektoren**.

## Was machen CSS-Selektoren?

Selektoren legen fest, **welche HTML-Elemente** von einer Gestaltung betroffen sein sollen.

Sie sind damit die Verbindung zwischen

- der Struktur in HTML
- und der Darstellung in CSS.

## Warum sind Selektoren so wichtig?

Ohne Selektoren könnte man Stile nicht gezielt anwenden. Erst durch sie wird steuerbar,

- welches Element ein bestimmtes Aussehen bekommt,
- welche Bereiche gemeinsam gestaltet werden,
- und wie präzise Gestaltung auf einer Seite wirkt.

## HTML und CSS greifen hier zusammen

Selektoren zeigen besonders deutlich, warum Web Development nicht nur aus isolierten Einzelteilen besteht:

- HTML liefert die Struktur,
- CSS beschreibt die Gestaltung,
- Selektoren verbinden beides miteinander.

## Was du zuerst verstehen solltest

Am Anfang ist besonders wichtig:

- Selektoren als Auswahlmechanismus zu lesen,
- ihre Rolle zwischen Struktur und Gestaltung zu verstehen,
- nicht sofort nur an Syntax, sondern zuerst an Funktion zu denken.

## Merksatz

CSS-Selektoren bestimmen, welche HTML-Elemente gestaltet werden, und gehören damit zu den zentralen Grundideen von CSS.
