# Stichprobe und Grundgesamtheit

Die Grundgesamtheit ist die Menge, über die eine Aussage gemacht werden soll. Eine Stichprobe ist der tatsächlich untersuchte Teil davon.

## Einordnung

Die Unterscheidung ist wichtig, weil Daten nicht automatisch repräsentativ sind. Eine große, falsch ausgewählte Stichprobe kann verzerrt sein.

## Zentrale Aspekte

- **Zielmenge:** Zuerst muss klar sein, worüber gesprochen wird.
- **Auswahlverfahren:** Die Auswahl beeinflusst Verzerrungen.
- **Stichprobengröße:** Mehr Daten senken Zufallsschwankungen.
- **Repräsentativität:** Relevante Merkmale sollen angemessen abgebildet sein.

## Beispiele und Zusammenhang

Eine Umfrage in einer einzigen Klasse erlaubt nur vorsichtige Aussagen über alle Jugendlichen.

## Abgrenzung

Eine Stichprobe ist nicht automatisch repräsentativ, nur weil sie groß ist.

## Häufige Missverständnisse

Aus bequem erhobenen Daten wird oft auf eine viel größere Gruppe geschlossen.

## Kurz zusammengefasst

- Grundgesamtheit und Stichprobe müssen getrennt werden.
- Auswahlverfahren beeinflussen Aussagekraft.
- Größe ersetzt keine Repräsentativität.
