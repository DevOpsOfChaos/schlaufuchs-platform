# Lösung: Proportionale Modelle – Aufgabe A

Eine mögliche Lösung ist:

1. **Gegebenes sammeln:** Zuerst werden alle Zahlen, Bedingungen und Begriffe aus der Situation notiert.
2. **Gesuchte Aussage festlegen:** Danach wird klar formuliert, ob ein Wert, ein Vergleich, eine Begründung oder eine Deutung gesucht ist.
3. **Passenden Weg wählen:** Für **Proportionale Modelle** wird ein Rechenweg oder eine Darstellung gewählt, die zur Struktur der Aufgabe passt.
4. **Kontrolle durchführen:** Das Ergebnis wird über Einheit, Größenordnung, Randfall oder Gegenprobe geprüft.

## Beispiel für einen Antwortsatz

„Das Ergebnis ist plausibel, weil es zur gegebenen Größenordnung passt und die verwendete Beziehung die Situation sinnvoll beschreibt.“

## Warum das reicht

Die Lösung zeigt nicht nur einen Rechenansatz, sondern auch, wie man erkennt, ob das Ergebnis fachlich tragfähig ist.
