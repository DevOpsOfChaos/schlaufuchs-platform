## Aufgabe 1: Struktur markieren

Der gesamte Block ist ein Objekt. <code>name</code> ist ein Schlüssel. <code>["Lesen", "Klettern"]</code> ist ein Array. <code>true</code> ist ein Bool-Wert.

## Aufgabe 2: Verschachtelung erklären

JSON erlaubt als Wert nicht nur einfache Inhalte, sondern auch Arrays oder weitere Objekte. Dadurch lassen sich Hierarchien ausdrücken.

## Aufgabe 3: Denkfehler prüfen

<code>"true"</code> ist eine Zeichenkette. <code>true</code> ist ein boolescher Wert. Beides ist in JSON nicht dasselbe.
