## Aufgabe 1: Website aufrufen

- **Client**: der Browser auf dem eigenen Gerät  
- **Server**: der Rechner oder Dienst, der die Website bereitstellt  
- **Anfrage**: Der Browser fordert die Inhalte der Website an.  
- **Antwort**: Der Server sendet die angeforderten Daten zurück, und der Browser zeigt sie auf dem Bildschirm an.

Eine mögliche Formulierung:  
„Beim Aufruf einer Website schickt der Browser als Client eine Anfrage an einen Server. Der Server verarbeitet diese Anfrage und sendet die passenden Inhalte zurück. Die Antwort wird danach im Browser sichtbar.“

## Aufgabe 2: Rollen prüfen

„Der Server ist einfach nur das Internet“ ist zu ungenau, weil das Internet eher das Netz und die Verbindungen beschreibt.  
Ein Server ist ein konkreter Dienst oder Rechner, der Anfragen entgegennimmt und Inhalte oder Funktionen bereitstellt.

## Aufgabe 3: Übertragen

Beispiel Messenger:

1. Die App sendet eine Nachricht ab.
2. Die App ist dabei der Client.
3. Ein Server nimmt die Nachricht an und verarbeitet sie.
4. Der Server leitet sie an das richtige Ziel weiter.
5. Auf dem anderen Gerät erscheint die Nachricht.

## Worauf man bei der Korrektur achten sollte

- Client und Server klar trennen.
- Anfrage und sichtbare Antwort beide nennen.
- Beim Alltagsbeispiel einen nachvollziehbaren Ablauf statt bloßer Schlagworte geben.
