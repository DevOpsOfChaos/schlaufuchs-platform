## Aufgabe 1

Verarbeitet wird serverseitig nur der Bereich innerhalb der PHP-Tags. HTML bleibt als Strukturteil der Antwort erhalten. Kommentare erklären den Code nur und erzeugen keine sichtbare Ausgabe.

## Aufgabe 2

Der Kommentar ist für Menschen gedacht und wird nicht als normaler sichtbarer Seiteninhalt ausgegeben.

## Aufgabe 3

Weil PHP-Tags verarbeitbaren Servercode markieren, HTML-Tags aber Struktur im Ergebnis beschreiben. Beide haben also unterschiedliche Rollen.
