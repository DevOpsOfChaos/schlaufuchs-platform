## Musterloesung

### Aufgabe 1

Die aeussere Ebene sammelt mehrere Personendatensaetze in einer gemeinsamen Struktur. Die innere Ebene beschreibt dann jeweils die einzelnen Eigenschaften einer bestimmten Person.

### Aufgabe 2

Die innere Ebene stellt den eigentlichen Einzel-Datensatz bereit, also zum Beispiel Name und Stadt einer Person. Sie liefert damit die Details zu einem Element der aeusseren Sammlung.

### Aufgabe 3

Mehrdimensionale Arrays sind nicht nur viele Klammern, sondern eine klar gegliederte Datenstruktur. Erst die Trennung der Ebenen macht die enthaltenen Datensaetze ruhig lesbar.
