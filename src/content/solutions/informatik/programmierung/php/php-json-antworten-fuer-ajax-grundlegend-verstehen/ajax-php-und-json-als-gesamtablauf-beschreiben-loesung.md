## Aufgabe 1

JavaScript startet im Browser eine Anfrage im Hintergrund. Das PHP-Skript verarbeitet die Anforderung und gibt strukturierte Daten als JSON zurück. Danach liest JavaScript die Antwort und aktualisiert gezielt nur den betroffenen Seitenteil.

## Aufgabe 2

Weil JSON strukturierte Daten mit Schlüsseln und Werten liefert. JavaScript kann solche Daten leichter fachlich lesen und einzelnen Oberflächenelementen zuordnen als bloßen unstrukturierten Text.
