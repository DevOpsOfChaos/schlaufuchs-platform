## Aufgabe 1

Die Funktion kann funktionieren, weil der dritte Parameter einen Defaultwert besitzt. Wenn er im Aufruf fehlt, wird der vorgegebene Standard verwendet.

## Aufgabe 2

Optionale Parameter stehen am Ende, damit zuerst alle Pflichtargumente eindeutig zugeordnet werden können.
