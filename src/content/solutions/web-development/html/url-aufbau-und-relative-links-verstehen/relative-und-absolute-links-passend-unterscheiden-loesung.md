## Aufgabe 1: Linkart wählen

- interne Projektdatei → eher relativ
- externe Dokumentation → eher absolut
- Sprung im selben Dokument → Anker mit <code>#...</code>

## Aufgabe 2: Anker erklären

<code>#kontakt</code> verweist auf eine markierte Stelle im aktuellen Dokument.

## Aufgabe 3: Projektlogik

Relative Verlinkung ist oft robuster, wenn ein Projekt als Ganzes in ein anderes Verzeichnis oder auf einen anderen Server umzieht.
