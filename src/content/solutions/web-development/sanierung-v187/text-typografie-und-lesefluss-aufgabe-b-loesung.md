## Musterlösung zu `web-v187-004b`

    Ausgangsauftrag: Leite aus einem unruhigen Textbild drei CSS-Regeln ab, die den Lesefluss verbessern.

    Eine mögliche Lösung ist:

    1. Der Zweck wird zuerst auf Lesbare Texte im Web eingegrenzt.
2. Die beteiligten Inhalte oder Dateien werden sichtbar getrennt.
3. Jede Entscheidung bekommt eine kurze Begründung statt nur eine optische Wirkung.
4. Eine typische Fehlerquelle wird geprüft und vermieden.
5. Die überarbeitete Fassung nennt zusätzlich, was vorher unklar war und warum die neue Ordnung stabiler ist.

    ## Beispielhafte Auswertung

    - **Rolle:** Lesbare Texte im Web
    - **wichtiger Kontrollpunkt:** Ein guter Webtext bleibt auch ohne Design verständlich und wird durch CSS nur ruhiger lesbar gemacht.
    - **häufige Fehlerquelle:** Fettung ersetzt eine echte Überschrift.

    ## Begründung

    Die Lösung ist ausreichend, wenn sie nicht nur das Endergebnis nennt, sondern die Entscheidung sichtbar macht. Bei Web-Development-Aufgaben zählt besonders, ob HTML-Struktur, CSS-Gestaltung, JavaScript-Verhalten und Projektorganisation sauber getrennt wurden.
