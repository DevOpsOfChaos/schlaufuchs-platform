## Aufgabe 1: Layout wählen

Grid liegt näher, weil mehrere Bereiche gleichzeitig als Fläche mit Zeilen und Spalten angeordnet werden sollen: oben Header, links Navigation, rechts Inhalt, unten Footer.

## Aufgabe 2: Leiste wählen

Flexbox liegt näher, weil hier vor allem eine Richtung wichtig ist: die horizontale Anordnung der Buttons.

## Aufgabe 3: Fläche oder Richtung?

Ein Ein-Richtungs-Layout ordnet Elemente hauptsächlich entlang einer Achse. Ein Flächenlayout organisiert Bereiche gleichzeitig in Zeilen und Spalten.

## Worauf man bei der Korrektur achten sollte

- Grid nicht einfach als "moderner" beschreiben, sondern passend zum Flächenproblem begründen.
- Flexbox nicht abwerten, sondern als passende Ein-Richtungs-Lösung einordnen.
