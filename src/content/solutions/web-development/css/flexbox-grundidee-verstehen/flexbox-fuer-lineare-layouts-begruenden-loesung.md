## Aufgabe 1: Anwendungsfall

Flexbox passt hier gut, weil die Buttons hauptsächlich entlang einer Richtung verteilt werden sollen: nebeneinander in einer Reihe.

## Aufgabe 2: Containerfrage

Der gemeinsame Container steuert die Verteilung seiner direkten Kinder. Würde man nur einzelne Buttons formatieren, entstünde keine gemeinsame Flex-Verteilung.

## Aufgabe 3: Richtung statt Fläche

Ein lineares Layout denkt vor allem in einer Richtung, ein Flächenlayout gleichzeitig in Zeilen und Spalten.
