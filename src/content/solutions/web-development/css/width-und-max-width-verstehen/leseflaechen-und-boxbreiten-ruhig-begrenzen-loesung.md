## Musterlösung

### Aufgabe 1

Eine Box mit fester Breite ist starrer. Eine Box mit `width: 100%` und sinnvoller `max-width` passt sich auf kleinen Geräten besser an und bleibt auf großen Flächen trotzdem begrenzt.

### Aufgabe 2

`max-width` schützt auch die Lesbarkeit, weil Texte nicht unnötig breit werden. So müssen die Augen keine extrem langen Zeilen verfolgen.

### Aufgabe 3

Responsive heißt nicht einfach überall 100 Prozent. Oft geht es darum, flexibel zu wachsen, aber trotzdem eine sinnvolle Obergrenze zu behalten.
