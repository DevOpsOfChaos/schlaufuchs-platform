## Lösung

Eine Container Query passt, wenn dieselbe Komponente in unterschiedlich breiten Bereichen vorkommt. Dann zählt nicht die Fensterbreite, sondern der tatsächliche Platz der Komponente.

## Kontrollgedanke

Die Lösung ist gut, wenn sie nicht nur Syntax nennt, sondern die Verantwortung sauber trennt: Struktur lesen, Zustand entscheiden, verständlich rückmelden und den Fehlerfall nicht unsichtbar lassen.
