## Lösung

Der Warenkorb-Button als Kernaktion sollte sofort funktionieren. Eine große Bildergalerie kann teilweise später laden. Ein Bewertungsdialog kann sehr gut erst beim Öffnen nachgeladen werden, solange die Grundseite stabil bleibt.

## Kontrollgedanke

Die Lösung ist gut, wenn sie nicht nur Syntax nennt, sondern die Verantwortung sauber trennt: vorhandene Struktur lesen, Entscheidung treffen, Rückmeldung geben und Fehler nicht unsichtbar lassen.
