## Lösung

Ein leerer Wert soll nicht suchen, ein zu kurzer Wert soll eine verständliche Meldung zeigen, ein gültiger Wert darf die Suche starten. Diese Logik kann als kleine Funktion geprüft werden.

## Kontrollgedanke

Die Lösung ist gut, wenn sie nicht nur Syntax nennt, sondern die Verantwortung sauber trennt: Struktur lesen, Zustand entscheiden, verständlich rückmelden und den Fehlerfall nicht unsichtbar lassen.
