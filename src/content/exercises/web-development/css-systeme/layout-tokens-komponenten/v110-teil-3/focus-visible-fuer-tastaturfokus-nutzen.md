## Aufgabe

Bearbeite das Szenario zu **:focus-visible für Tastaturfokus nutzen**.

## Szenario

Fokusringe sollen sichtbar sein, ohne Mausnutzer zu irritieren.

## Arbeitsauftrag

1. Ordne die Situation dem genauen Unterthema zu.
2. Erkläre die naheliegende Verwechslung: Fokusmarkierungen werden komplett entfernt.
3. Nutze diesen Prüfschritt: Nutze :focus-visible statt outline: none.
4. Formuliere eine Gegenprobe, bei der sich ein Wert, Zustand oder Kontext ändert.

## Erwartete Abgabe

- eine klare Einordnung,
- eine fachliche Abgrenzung,
- ein konkreter Prüfschritt,
- eine kurze Gegenprobe.
