## Aufgabe

Bearbeite das Szenario zu **Stacking Context vor z-index-Erhöhung prüfen**.

## Szenario

Ein Dropdown liegt trotz hohem z-index unter einem anderen Element.

## Arbeitsauftrag

1. Ordne die Situation dem genauen Unterthema zu.
2. Erkläre die naheliegende Verwechslung: Der z-index wird immer weiter erhöht.
3. Nutze diesen Prüfschritt: Prüfe, ob ein neuer Stacking Context die Reihenfolge begrenzt.
4. Formuliere eine Gegenprobe, bei der sich ein Wert, Zustand oder Kontext ändert.

## Erwartete Abgabe

- eine klare Einordnung,
- eine fachliche Abgrenzung,
- ein konkreter Prüfschritt,
- eine kurze Gegenprobe.
