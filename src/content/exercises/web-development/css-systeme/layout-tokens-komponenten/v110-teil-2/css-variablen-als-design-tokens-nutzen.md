## Aufgabe

Bearbeite das Szenario zu **CSS-Variablen als Design Tokens nutzen**.

## Szenario

Farben und Abstände sollen an vielen Stellen gleich bleiben.

## Arbeitsauftrag

1. Ordne die Situation dem genauen Unterthema zu.
2. Erkläre die naheliegende Verwechslung: Werte werden überall direkt kopiert.
3. Nutze diesen Prüfschritt: Lege zentrale Tokens fest und nutze sie in Komponenten.
4. Formuliere eine Gegenprobe, bei der sich ein Wert, Zustand oder Kontext ändert.

## Erwartete Abgabe

- eine klare Einordnung,
- eine fachliche Abgrenzung,
- ein konkreter Prüfschritt,
- eine kurze Gegenprobe.
