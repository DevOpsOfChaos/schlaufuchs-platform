## Aufgabe

Bearbeite das Szenario zu **Container Queries statt nur Viewport denken**.

## Szenario

Eine Karte steht mal in einer Sidebar und mal im Hauptbereich.

## Arbeitsauftrag

1. Ordne die Situation dem genauen Unterthema zu.
2. Erkläre die naheliegende Verwechslung: Responsive Design wird nur über die Fensterbreite gesteuert.
3. Nutze diesen Prüfschritt: Frage, wie breit der Container der Komponente ist.
4. Formuliere eine Gegenprobe, bei der sich ein Wert, Zustand oder Kontext ändert.

## Erwartete Abgabe

- eine klare Einordnung,
- eine fachliche Abgrenzung,
- ein konkreter Prüfschritt,
- eine kurze Gegenprobe.
