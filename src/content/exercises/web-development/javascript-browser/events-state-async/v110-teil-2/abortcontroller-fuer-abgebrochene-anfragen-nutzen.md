## Aufgabe

Bearbeite das Szenario zu **AbortController für abgebrochene Anfragen nutzen**.

## Szenario

Eine Suche startet neue Requests, während alte noch laufen.

## Arbeitsauftrag

1. Ordne die Situation dem genauen Unterthema zu.
2. Erkläre die naheliegende Verwechslung: Alte Antworten dürfen unkontrolliert neue Ergebnisse überschreiben.
3. Nutze diesen Prüfschritt: Brich überholte Anfragen mit AbortController ab.
4. Formuliere eine Gegenprobe, bei der sich ein Wert, Zustand oder Kontext ändert.

## Erwartete Abgabe

- eine klare Einordnung,
- eine fachliche Abgrenzung,
- ein konkreter Prüfschritt,
- eine kurze Gegenprobe.
