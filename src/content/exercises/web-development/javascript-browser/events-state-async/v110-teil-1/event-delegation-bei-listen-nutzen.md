## Aufgabe

Bearbeite das Szenario zu **Event Delegation bei Listen nutzen**.

## Szenario

Eine Liste bekommt dynamisch neue Einträge.

## Arbeitsauftrag

1. Ordne die Situation dem genauen Unterthema zu.
2. Erkläre die naheliegende Verwechslung: Jeder Eintrag erhält einzeln denselben Listener.
3. Nutze diesen Prüfschritt: Hänge den Listener an ein gemeinsames Elternelement.
4. Formuliere eine Gegenprobe, bei der sich ein Wert, Zustand oder Kontext ändert.

## Erwartete Abgabe

- eine klare Einordnung,
- eine fachliche Abgrenzung,
- ein konkreter Prüfschritt,
- eine kurze Gegenprobe.
