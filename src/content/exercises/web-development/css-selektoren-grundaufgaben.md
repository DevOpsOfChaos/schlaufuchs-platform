Diese Aufgabe hält die Grundidee von CSS-Selektoren bewusst einfach: Du sollst verstehen, **wie Gestaltung gezielt an HTML-Strukturen gebunden wird**.

## Lernziel

Nach dieser Aufgabe solltest du sicher einordnen können,

- warum CSS eine Auswahl von Elementen braucht,
- welche Rolle Selektoren zwischen HTML und CSS spielen,
- warum gezielte Auswahl für Webgestaltung so wichtig ist.

## Aufgabe 1 – Was wird ausgewählt?

Beschreibe bei einfachen Beispielen,

- welches HTML-Element angesprochen wird,
- warum gerade dieses Element gestaltet werden soll,
- was der Selektor damit fachlich leistet.

## Aufgabe 2 – Struktur und Gestaltung verbinden

Erkläre in eigenen Worten,

- warum HTML allein keine Gestaltung festlegt,
- warum CSS ohne Selektoren nicht zielgerichtet arbeiten kann,
- wie beide Ebenen zusammenwirken.

## Aufgabe 3 – Präzision verstehen

Beschreibe, warum es im Web wichtig ist, Gestaltung nicht wahllos, sondern gezielt auf bestimmte Elemente anzuwenden.

Nutze dabei Begriffe wie Struktur, Auswahl, Gestaltung oder Verknüpfung.

## Aufgabe 4 – Aussagen prüfen

Ordne als **richtig** oder **falsch** ein:

- Selektoren helfen CSS dabei, bestimmte HTML-Elemente auszuwählen.
- Selektoren gehören nur zu HTML, nicht zu CSS.
- HTML und CSS werden über Selektoren sinnvoll miteinander verbunden.
- Ohne Auswahlregeln wäre gezielte Gestaltung schwer möglich.

Begründe kurz.

## Hinweise

- Denke Selektoren zuerst als Auswahlidee, nicht nur als Schreibweise.
- Achte darauf, dass Struktur und Gestaltung unterschiedliche Aufgaben haben.
- Gute Webentwicklung verbindet beide Ebenen gezielt.

## Selbstkontrolle

Du bist auf einem guten Stand, wenn du am Ende sicher sagen kannst,

- warum Selektoren gebraucht werden,
- wie sie HTML und CSS verbinden,
- warum gezielte Auswahl für Webgestaltung zentral ist.
