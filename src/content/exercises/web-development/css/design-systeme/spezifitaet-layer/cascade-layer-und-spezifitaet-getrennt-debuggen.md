## Aufgabe

Bearbeite das Szenario zu **Cascade Layer und Spezifität**.

## Szenario

Eine CSS-Regel greift trotz scheinbar stärkerem Selektor nicht.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Prüfe erst Layer, dann Selektorstärke.
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
