## Aufgabe 1: Ergänzung oder Konflikt?

Beurteile diese beiden Kombinationen:

1. <code>.karte { padding: 1rem; }</code> und <code>.warnung { color: red; }</code>
2. <code>.karte { color: black; }</code> und <code>.warnung { color: red; }</code>

Erkläre jeweils, ob die Regeln sich ergänzen oder konkurrieren.

## Aufgabe 2: Gewinner begründen

Warum ist bei konkurrierenden Regeln nicht jede doppelte Angabe automatisch ein Fehler, sondern eine Kaskadenfrage?
