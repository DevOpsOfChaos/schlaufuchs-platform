## Aufgabe

Bearbeite das Szenario zu **min-width und max-width**.

## Szenario

Eine Karte soll wachsen, aber nicht zu breit werden.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Prüfe kleinste erlaubte und größte erlaubte Breite.
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
