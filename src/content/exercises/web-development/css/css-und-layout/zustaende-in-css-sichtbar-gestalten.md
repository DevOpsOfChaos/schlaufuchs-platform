## Aufgabe

Bearbeite die Situation:

> Ein Button kann hover, focus und disabled sein.

1. Ordne die Situation dem kleinen Unterthema **CSS Layout und Gestaltung** zu.
2. Erkläre, warum die naheliegende Verwechslung problematisch ist: Nur der Normalzustand wird gestaltet.
3. Formuliere einen konkreten Prüfschritt.
4. Schreibe eine kurze Gegenprobe mit einem veränderten Wert, Zustand oder Beispiel.
