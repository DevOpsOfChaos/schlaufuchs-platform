## Aufgabe

Bearbeite das Szenario zu **Flexbox und Grid**.

## Szenario

Ein Layout braucht entweder eine Zeile mit flexiblen Elementen oder ein zweidimensionales Raster.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Frage: eine Achse oder zwei Achsen?
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
