## Aufgabe

Bearbeite das Szenario zu **Event Delegation begründen**.

> Listeneinträge werden nachgeladen.

1. Beschreibe zuerst, welche konkrete Entscheidung getroffen werden muss.
2. Nutze die Prüffrage: **Kann ein gemeinsamer Elternknoten das Ereignis auswerten?**

4. Formuliere eine Gegenprobe: Was müsste sich ändern, damit deine Entscheidung nicht mehr passt?
