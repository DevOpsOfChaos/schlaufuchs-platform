## Aufgabe

Bearbeite das Szenario zu **Event Delegation**.

## Szenario

Listeneinträge werden später dynamisch ergänzt.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Hänge den Listener an ein stabiles Elternelement.
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
