## Aufgabe

Bearbeite das Szenario zu **Fetch-Fehlerarten**.

## Szenario

Eine Anfrage erhält 404 oder bricht wegen Netzwerkfehler ab.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Prüfe response.ok getrennt vom Netzwerkfehler.
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
