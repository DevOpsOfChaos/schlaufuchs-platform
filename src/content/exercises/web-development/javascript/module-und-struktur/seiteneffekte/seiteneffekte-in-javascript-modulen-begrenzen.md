## Aufgabe

Bearbeite ein kleines Szenario zu **Modulgrenzen**.

1. Benenne das genaue Unterthema.
2. Nenne eine typische Verwechslung.
3. Löse den Fall mit einem passenden Prüfschritt.
4. Formuliere eine Gegenprobe.

## Szenario

In einer Übung taucht genau dieses Thema auf. Eine breite Antwort klingt möglich, würde aber zu ungenau bleiben. Entscheide deshalb, welche konkrete Trennkante du zuerst prüfen musst.

## Erwartete Abgabe

- Einordnung in einem Satz,
- kurzer Lösungsweg,
- Gegenprobe,
- Abgrenzung vom nächstbreiteren Thema.
