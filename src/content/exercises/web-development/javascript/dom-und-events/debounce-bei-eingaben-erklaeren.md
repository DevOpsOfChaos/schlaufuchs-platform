## Aufgabe

Bearbeite die Situation:

> Eine Suche reagiert auf jedes Tastaturereignis.

1. Ordne die Situation dem kleinen Unterthema **JavaScript DOM und Events** zu.
2. Erkläre, warum die naheliegende Verwechslung problematisch ist: Jeder Tastendruck startet sofort teure Arbeit.
3. Formuliere einen konkreten Prüfschritt.
4. Schreibe eine kurze Gegenprobe mit einem veränderten Wert, Zustand oder Beispiel.
