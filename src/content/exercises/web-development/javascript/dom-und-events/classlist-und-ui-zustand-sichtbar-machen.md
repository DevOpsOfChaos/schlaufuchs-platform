## Aufgabe

Plane drei Zustandsklassen für eine Hinweisbox: neutral, Erfolg und Fehler. Erkläre, was JavaScript und was CSS übernimmt.

## Bearbeitungshinweise

1. Benenne zuerst die sichtbare Situation auf der Seite.
2. Trenne Auswahl, Ereignis, Daten und Rückmeldung voneinander.
3. Schreibe keine große Komplettlösung auf einmal, sondern begründe die Codeentscheidung Schritt für Schritt.
4. Prüfe am Ende, welcher Fehler- oder Leerzustand auftreten könnte.

## Erwartetes Ergebnis

Deine Lösung soll nicht nur Code enthalten, sondern auch erklären, warum dieser Code für die Browser-Situation passend ist.
