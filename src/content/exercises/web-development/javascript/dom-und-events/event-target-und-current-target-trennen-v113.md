## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze diesen Prüfschritt: **Welches Element hat den Handler und welches wurde wirklich geklickt?**
