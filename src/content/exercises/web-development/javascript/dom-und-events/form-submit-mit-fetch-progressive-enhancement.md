## Aufgabe

Skizziere ein Formular, das ohne JavaScript abschickbar bleibt und mit JavaScript eine bessere Statusmeldung zeigt.

## Arbeitsauftrag

1. Beschreibe zuerst die Ausgangssituation in zwei bis drei Sätzen.
2. Markiere, welcher Teil zu HTML, CSS oder JavaScript gehört.
3. Formuliere eine robuste Lösungsidee mit sichtbarer Rückmeldung.
4. Prüfe am Ende, ob die Lösung auch bei Fehlern, Tastaturbedienung oder fehlenden Daten nachvollziehbar bleibt.
