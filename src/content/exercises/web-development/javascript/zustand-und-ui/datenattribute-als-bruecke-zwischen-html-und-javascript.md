## Aufgabe

Entwirf drei Filter-Buttons für Aufgaben: alle, offen, erledigt. Nutze data-Attribute und erkläre, warum das lesbarer ist als Styling-Klassen.

## Arbeitsauftrag

1. Markiere zuerst, welche HTML-Struktur oder welcher Zustand vorhanden ist.
2. Beschreibe dann, welche JavaScript-Entscheidung nötig ist.
3. Formuliere eine kurze Begründung für Nutzerführung, Robustheit oder Sicherheit.
4. Prüfe am Ende, ob dein Vorschlag auch bei Fehlern oder fehlenden Elementen verständlich bleibt.
