## Aufgabe

Bearbeite das Szenario zu **Button und Link**.

## Szenario

Ein Element löst entweder Navigation oder eine Aktion aus.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Frage: Navigieren oder Aktion ausführen?
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
