## Aufgabe

Verbessere eine Fehlermeldung für ein Pflichtfeld so, dass sie konkret und mit dem Feld verbunden ist.

Arbeite in drei Schritten:

1. Beschreibe zuerst die Situation aus Nutzersicht.
2. Benenne dann die technische Zuständigkeit.
3. Formuliere am Ende eine konkrete Verbesserung, die sichtbar oder prüfbar ist.

## Zusatzfrage

Welche Fehlannahme könnte in diesem Thema leicht entstehen, wenn man nur auf die Optik oder nur auf den Code schaut?
