## Aufgabe 1: Reihenfolge

Ordne diese Bausteine sinnvoll:

- <code>&lt;body&gt;</code>
- <code>&lt;title&gt;</code>
- <code>&lt;head&gt;</code>
- <code>&lt;!DOCTYPE html&gt;</code>
- <code>&lt;html lang="de"&gt;</code>
