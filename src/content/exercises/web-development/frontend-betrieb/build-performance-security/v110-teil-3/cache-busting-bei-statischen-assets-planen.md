## Aufgabe

Bearbeite das Szenario zu **Cache Busting bei statischen Assets planen**.

## Szenario

Eine neue CSS-Datei ist deployed, Nutzer sehen aber alte Styles.

## Arbeitsauftrag

1. Ordne die Situation dem genauen Unterthema zu.
2. Erkläre die naheliegende Verwechslung: Browsercache wird als zufälliges Problem behandelt.
3. Nutze diesen Prüfschritt: Nutze versionierte Dateinamen oder Hashes im Build.
4. Formuliere eine Gegenprobe, bei der sich ein Wert, Zustand oder Kontext ändert.

## Erwartete Abgabe

- eine klare Einordnung,
- eine fachliche Abgrenzung,
- ein konkreter Prüfschritt,
- eine kurze Gegenprobe.
