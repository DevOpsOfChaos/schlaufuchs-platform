## Aufgabe

Betrachte ein einzelnes Element mit Text, Innenabstand, Rahmen und Abstand zum nächsten Element.

Ordne zu,

- was der Inhalt ist,
- wo padding liegt,
- welche Rolle border spielt,
- und was margin von den inneren Schichten unterscheidet.
