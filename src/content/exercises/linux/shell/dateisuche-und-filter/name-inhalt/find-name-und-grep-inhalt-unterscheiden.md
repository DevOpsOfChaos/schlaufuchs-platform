## Aufgabe

Bearbeite das Szenario zu **Dateiname und Dateiinhalt**.

## Szenario

Eine Datei soll nach Name oder nach Text gefunden werden.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Frage: Suchst du den Namen oder den Inhalt?
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
