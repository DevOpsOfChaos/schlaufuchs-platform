## Aufgabe

Bearbeite das Szenario zu **Public und Private Key**.

## Szenario

Ein SSH-Schlüssel soll auf einem Server eingerichtet werden.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Nur der öffentliche Schlüssel gehört auf den Server.
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
