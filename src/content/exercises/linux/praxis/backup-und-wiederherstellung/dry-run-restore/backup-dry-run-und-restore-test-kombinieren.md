## Aufgabe

Bearbeite das Szenario zu **Dry-run und Restore-Test**.

## Szenario

Ein Backup-Skript soll sicherer werden.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Plane Dry-run und echte Wiederherstellungsprobe.
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
