## Aufgabe

Bearbeite das Szenario zu **systemctl start und enable**.

## Szenario

Ein Dienst läuft jetzt, soll aber auch nach Neustart laufen.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Prüfe Laufzustand und Autostart getrennt.
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
