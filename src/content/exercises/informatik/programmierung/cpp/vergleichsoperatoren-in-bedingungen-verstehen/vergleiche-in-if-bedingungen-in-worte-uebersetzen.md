## Aufgabe 1: In Worte übersetzen

Übersetze diese Bedingung in ruhige Alltagssprache:

```cpp
if (punkte >= 50)
```

## Aufgabe 2: Vergleich erklären

Erkläre den Unterschied zwischen:

- <code>x == 5</code>
- <code>x != 5</code>

## Aufgabe 3: Denkfehler prüfen

Jemand schreibt in einer Bedingung:

```cpp
if (x = 5)
```

Erkläre, warum das fachlich nicht dieselbe Aussage ist wie <code>x == 5</code>.
