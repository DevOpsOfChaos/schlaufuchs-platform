## Aufgabe 1: Bereiche zuordnen

Ein Beispiel enthält:

- eine HTML-Überschrift,
- einen PHP-Block mit <code>echo</code>,
- einen einzeiligen Kommentar,
- einen abschließenden Absatz in HTML.

Erkläre kurz, welche Teile serverseitig verarbeitet werden und welche am Ende als HTML übrig bleiben.
