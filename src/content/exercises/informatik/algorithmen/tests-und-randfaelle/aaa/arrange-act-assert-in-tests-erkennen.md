## Aufgabe

Bearbeite das Szenario zu **Arrange, Act, Assert**.

## Szenario

Ein Test legt Daten an, führt Code aus und prüft ein Ergebnis.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Gliedere Testdaten, Aktion und Erwartung.
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
