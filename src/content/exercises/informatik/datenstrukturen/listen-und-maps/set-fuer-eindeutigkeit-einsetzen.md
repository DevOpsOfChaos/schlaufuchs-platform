## Aufgabe

2. Erkläre, warum die naheliegende Verwechslung problematisch ist: Dubletten werden später mühsam herausgefiltert.

4. Schreibe eine kurze Gegenprobe mit einem veränderten Wert, Zustand oder Beispiel.
