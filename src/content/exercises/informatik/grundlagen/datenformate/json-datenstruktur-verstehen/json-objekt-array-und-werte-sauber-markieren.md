## Aufgabe 1: Struktur markieren

Gegeben ist:

```json
{
  "name": "Mira",
  "hobbys": ["Lesen", "Klettern"],
  "newsletter": true
}
```

Benenne:
- das Objekt,
- den Schlüssel `name`,
- das Array,
- und den Bool-Wert.

## Aufgabe 2: Verschachtelung erklären

Beschreibe, warum ein Wert in JSON selbst wieder ein Objekt oder ein Array sein kann.

## Aufgabe 3: Denkfehler prüfen

Jemand schreibt `"true"` und `true` seien in JSON dasselbe. Erkläre den Fehler.
