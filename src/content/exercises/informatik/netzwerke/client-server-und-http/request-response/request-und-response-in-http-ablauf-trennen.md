## Aufgabe

Bearbeite das Szenario zu **HTTP Request und Response**.

## Szenario

Ein Browser ruft eine Seite ab und erhält Daten zurück.

## Arbeitsauftrag

1. Benenne das genaue Unterthema in einem Satz.
2. Erkläre, welche Verwechslung hier naheliegt.
3. Nutze den passenden Prüfschritt: Markiere Richtung, Sender und Inhalt jeder Nachricht.
4. Formuliere eine Gegenprobe mit einem veränderten Wert, Zustand oder Kontext.

## Erwartete Abgabe

- kurze Einordnung,
- eine fachliche Begründung,
- ein konkreter Prüfschritt,
- eine Gegenprobe.
