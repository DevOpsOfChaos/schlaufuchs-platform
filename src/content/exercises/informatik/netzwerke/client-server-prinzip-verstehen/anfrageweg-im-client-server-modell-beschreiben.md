## Aufgabe 1: Website aufrufen

Beschreibe den Ablauf, wenn jemand eine Website im Browser öffnet.

Gehe dabei auf diese Punkte ein:

- Was ist der Client?
- Was ist der Server?
- Was wird angefragt?
- Wo erscheint die Antwort?

## Aufgabe 2: Rollen prüfen

Jemand sagt: „Der Server ist einfach nur das Internet.“

Erkläre, warum diese Aussage zu ungenau ist.

## Aufgabe 3: Übertragen

Nenne ein zweites Beispiel aus dem Alltag, das ebenfalls nach dem Client-Server-Prinzip funktioniert, und erkläre den Weg der Anfrage in 3 bis 5 Schritten.
