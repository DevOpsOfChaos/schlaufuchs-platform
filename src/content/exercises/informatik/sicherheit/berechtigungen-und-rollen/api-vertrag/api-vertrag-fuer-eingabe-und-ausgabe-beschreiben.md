## Aufgabe

Bearbeite den Fall zu **API-Vertrag**.

### Fall

Beschreibe einen Endpunkt für eine Suchfunktion mit Parametern, Ergebnis und Fehlerfall.

### Arbeitsauftrag

1. Ordne den Fall dem genauen Unterthema zu.
2. Nenne die naheliegende Verwechslung.
3. Führe den passenden Prüfschritt aus.
4. Formuliere eine Gegenprobe.

### Erwartete Abgabe

- ein kurzer Einordnungssatz,
- der konkrete Prüfschritt,
- eine knappe Begründung,
- eine Gegenprobe mit verändertem Wert, Zustand oder Kontext.
