## Aufgabe

Bearbeite das Szenario zu **Entprellung als Zeitproblem statt Code-Trick sehen**.

## Szenario

Ein Taster erzeugt beim Drücken mehrere schnelle Wechsel.

## Arbeitsauftrag

1. Ordne die Situation dem genauen Unterthema zu.
2. Erkläre die naheliegende Verwechslung: Mehrfaches Auslösen wird als Programmfehler allein verstanden.
3. Nutze diesen Prüfschritt: Betrachte das Signal über einige Millisekunden.
4. Formuliere eine Gegenprobe, bei der sich ein Wert, Zustand oder Kontext ändert.

## Erwartete Abgabe

- eine klare Einordnung,
- eine fachliche Abgrenzung,
- ein konkreter Prüfschritt,
- eine kurze Gegenprobe.
