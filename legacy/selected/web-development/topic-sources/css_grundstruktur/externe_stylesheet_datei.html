Die externe Stylesheet-Datei

Prinzipiell gibt es verschiedene Möglichkeiten, CSS-Formatanweisungen ins HTML-Dokument zu integrieren. Bei Websites, die häufig mehrere Unterseiten beinhalten, ist es sinnvoll, die CSS-Formatanweisungen in eine externe Datei auszulagern. Dies hat den Vorteil, dass sich Änderungen in dieser externen Stylesheet-Datei auf alle HTML-Dokumente bezieht, mit denen diese Datei verlinkt ist.

Externe Stylesheet-Datei ins HTML-Dokument einbinden

Die externe Stylesheet-Datei wird mit dem HTML-Element <link> im <head>-Bereich des HTML-Dokuments eingefügt.

Folgende Syntax wird dabei verwendet:

<link rel="stylesheet" href="CSS-StylesheetDateiName.css">

  • rel="stylesheet" gibt an, dass es sich um ein CSS-Stylesheet handelt.
  • href="CSS-StylesheetDateiName.css" definiert den Ablageort der Stylesheet-Datei. Dabei gelten die Regeln für Verweise auf Dokumente. Normalerweise steht hier ein relativer Verweis auf eine interne *.css Datei.
