# Linux-Lernshell – Funktionsstand

Dieser Sammelstand enthält eine universelle Linux-Lernshell, die bewusst mehr kann als ein enger Themen-Simulator.

## Schwerpunkt

- ruhige, erklärende Fehlertexte
- vorbereitete Szenarien pro Linux-Thema
- viele typische Grundbefehle
- Pipes und einfache Umleitungen
- Textarbeit, Dateiarbeit und einfache Systemabfragen

## Zusätzlich in diesem Stand

- `hostname`
- `uptime`
- `cal`
- `cmp`
- `sha256sum`
- `md5sum`

## Didaktische Idee

Die Shell soll breit genug sein, damit Lernen nicht künstlich wirkt, aber kontrolliert genug, damit keine echte Systemkomplexität die Seite kapert.
