# Repo Fachseiten Architecture Block 24

Dieser Block baut auf dem Fachseiten-Umbau aus Block 23 auf und schärft vor allem die Strukturqualität:

- Fach- und Fachthemenseiten nutzen jetzt fachspezifische Themenbeschreibungen statt generischer Listenfragmente.
- Themenkarten werden innerhalb eines Fachs bewusster sortiert und nicht nur alphabetisch ausgegeben.
- Auf Fachthemenseiten wird eine zusätzliche Unterebene nur noch dann gezeigt, wenn sie als eigener Themenblock wirklich sinnvoll ist.
- Die Bereichssuche erklärt jetzt sichtbar, dass Wissensseiten und Aufgaben in der Anzeige getrennt bleiben.

Ziel dieses Blocks:

- weniger Dashboard-Eindruck
- ruhigere Hierarchie
- verständlichere Themenblöcke
- weniger unnötige Zwischenseiten
