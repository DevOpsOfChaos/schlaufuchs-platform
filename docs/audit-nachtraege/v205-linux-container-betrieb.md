# Audit-Nachtrag v205: Linux Containerbetrieb und Observability

Dieser Block startet den nächsten großen Linux-Nachtrag in der neuen kumulativen Linie ab dem gepushten v196/main und baut auf v197 bis v204 auf.

## Umfang

- 72 neue Überblicksseiten
- 72 didaktische SVGs
- 144 Aufgaben
- 144 Lösungen

## Schwerpunkte

Container-Grundlagen, systemnaher Linux-Betrieb, Observability, Deployment, Sicherheit, Backup, Automation und Servicequalität.

## Prüffokus

- Jede Aufgabe besitzt eine Lösung.
- Jede Lösung verweist per `relatedExercise` auf die passende Aufgabe.
- Alle `taskId`-Werte sind eindeutig.
- SVGs sind parsebar.
- Frontmatter nutzt `subject: linux` und gültige Level.
- Umlaute bleiben UTF-8, keine Mojibake-Marker.
