# Audit-Nachtrag v197: Web Backend und API-Praxis

Neustart der kumulativen ZIP-Linie ab `main` nach v196.

## Umfang

- 16 Überblicksseiten
- 16 didaktische SVGs
- 32 Aufgaben
- 32 Lösungen

## Prüffokus

- Jede Aufgabe besitzt eine direkt passende Lösung.
- Jede Lösung enthält `taskId` und `relatedExercise`.
- `relatedExercise` verweist auf eine vorhandene Aufgabe ohne Dateiendung.
- Alle neuen Dateien sind additiv unter `sanierung-v197-neustart`.
- Keine bestehenden Projektdateien werden überschrieben.

## Inhaltliche Linie

Der Block stärkt Web Development mit Backend- und API-Grundlagen: Request/Response, Routen, Statuscodes, JSON, Fetch, Validierung, Sessions, CORS, Datenbankgrenzen, Sicherheit, Deployment, Debugging und API-Tests.
