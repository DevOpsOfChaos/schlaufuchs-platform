# Repo QA Polish Block 16

Dieser Block baut direkt auf `repo_polish_step15.zip` auf und bleibt kumulativ.

## Schwerpunkt

- ruhigere Startpunkte auf `wissen/` und `aufgaben/`
- stärkere Scanbarkeit auf Fachbereichsseiten
- klarere Schnellstarts in Themenpfaden
- Linux-Lernshell mit sichtbarerem Utility-Verhalten

## Konkrete Änderungen

### Wissen und Aufgaben

- pro Fachbereich eigene Einstiegskarten mit:
  - Anzahl der Einträge
  - erster direkter Seite/Aufgabe
  - Link zum Fachbereich
  - Link zum stärksten Themenpfad
- bestehende Schnellstart- und Fachblöcke bleiben erhalten

### Fachbereichsseiten

- neuer Block `Schnell starten`
- direkte Karten für:
  - erste Wissensseite
  - erste Aufgabe
  - stärksten Themenpfad

### Themenpfadseiten

- neue Hero-Kennzahlen für direkte Wissensseiten, Aufgaben und Unterthemen
- neuer Block `Gute erste Schritte in diesem Themenblock`
- benachbarte Themenpfade rücken weiter nach oben

### Linux-Lernshell

- Utility-Buttons für `help`, `pwd`, `history`
- Klick auf Vorschlagschips führt weiter direkt aus
- Statuszeile für Ort und Verlauf
- `Esc` leert die Eingabe
- `Ctrl + L` leert die Ausgabe
