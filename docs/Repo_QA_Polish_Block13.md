# Repo QA / Polish Block 13

## Ziel
Kleine Repo-basierte Qualitätsrunde nach dem erfolgreichen Build.

## Enthalten
- Wissensübersicht verlinkt Wissenskarten jetzt direkt auf `/wissen/...`.
- Fachbereichsübersichten verlinken direkte Wissenseinstiege ebenfalls direkt auf `/wissen/...`.
- Linux-Lernshell führt Vorschlagsbefehle per Klick direkt aus und erklärt den Verlauf sichtbarer.

## Wirkung
- Weniger unnötige Redirect-Hops über `/fachbereiche/.../<leaf>`.
- Ruhigeres Navigationsgefühl auf Übersichtsseiten.
- Schnellere Linux-Übungen ohne erst Chips in das Eingabefeld zu übernehmen.
