# Themenpfad-Korrekturen

In diesem Stand wurden zwei echte Kollisionen im Themenpfad bereinigt:

- Linux: `Benutzer und Gruppen – Grundlagen` -> eigener Überblickspfad `system/benutzer-und-gruppen-grundlagen`
- Elektrotechnik: `Kirchhoffsche Regeln tiefer verstehen` -> eigener Vertiefungspfad `elektronik/kirchhoffsche-regeln-tiefer-verstehen`

Damit entstehen im Bundle keine doppelten `(subject, topicPath)`-Ziele mehr.
