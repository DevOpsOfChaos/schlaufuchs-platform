# Informatik – Zielseite `programmieren-mit-c-und-python`

## Zweck dieser Datei
Diese Datei beschreibt die konkrete Zielseite `informatik/programmieren-mit-c-und-python` für den Relaunch.

Sie ist **kein Altseiten-Import**, sondern eine neue Produktspezifikation für eine verständliche Wissensseite zum Einstieg in Programmierung mit C und Python.

Die Seite soll zeigen, dass beide Sprachen genutzt werden können, um Probleme in Programme zu übersetzen – aber mit unterschiedlicher Nähe zur Hardware und unterschiedlicher Denkweise im Einstieg.

## Rolle der Seite im neuen Informatik-Aufbau
Die Seite gehört in den Abschnitt:
- Informatik
- Einstieg in Programmierung

Sie kommt sinnvoll **nach** den allgemeinen Informatik-Grundlagen und **vor** den spezifischen C-Seiten.

## Lernziel der Seite
Nach der Seite sollen Lernende verstehen:
- dass C und Python beide Programmiersprachen sind
- dass beide gemeinsame Grundideen teilen
- dass sie sich in Schreibweise und Denkstil unterscheiden können
- warum C im Altbestand als wichtiger Einstieg gesetzt ist

## Zielcharakter der neuen Seite
Die Seite soll:
- Gemeinsamkeiten und Unterschiede verständlich machen
- keine Sprachkrieg-Seite werden
- eine faire Einordnung von C und Python geben
- mit einem verpflichtenden Merksatz enden

## Empfohlener Seitenaufbau
## 1) Hero / Einstieg
- Titel: `Programmieren mit C und Python`
- kurzer Einleitungssatz
- 2–3 Lernziele

## 2) Abschnitt: Gemeinsame Grundideen
- Programme bestehen aus Anweisungen und Daten
- Eingaben, Verarbeitung und Ausgaben gibt es in beiden Welten
- logisches Denken ist wichtiger als die erste Syntax

## 3) Abschnitt: C als strukturierter, hardware-näherer Einstieg
- präziser
- stärker auf Details und Struktur sichtbar
- gut für Grundlagenverständnis

## 4) Abschnitt: Python als menschenfreundlicher Einstieg
- oft kürzere Syntax
- Fokus stärker auf Lesbarkeit und schnellerem Einstieg
- im Altbestand eher als Vergleichs- und Kontextsprache

## 5) Abschnitt: Unterschiede ohne Übertreibung
- gleiche Probleme, unterschiedliche Ausdrucksweise
- Sprache ist Werkzeug, Denklogik ist entscheidend

## 6) Lernmodus / interaktive Module
- C oder Python zuordnen
- Gemeinsamkeit oder Unterschied?
- kleine Mini-Beispiele lesen und vergleichen

## 7) Merksatz
> C und Python sind unterschiedliche Werkzeuge, mit denen dieselben Grundideen des Programmierens umgesetzt werden können.
