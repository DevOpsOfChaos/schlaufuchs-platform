# Overlay Release Notes – Block 8

## Was als großer Block neu dazugekommen ist

- konsolidierte Topics-Helfer statt verstreuter Themenlogik
- stärkere Fachbereichsseite mit Schwerpunkten und Themenpfad-Hotspots
- bessere `/wissen`- und `/aufgaben`-Übersichten
- sichere Catch-all-Fallbacks ohne kaputte Wissensseiten-URL
- neue QA-, Status- und Einspiel-Dokumente

## Warum dieser Block wichtig ist

Dieser Block macht den Overlay-Stand deutlich robuster für das spätere Einspielen. Er verschiebt den Schwerpunkt weg von reiner Inhaltsmenge hin zu:

- Navigationskonsistenz
- Routing-Sicherheit
- besserer Scanbarkeit
- klarerer Prüfbarkeit nach dem Einspielen

## Was danach sinnvoll ist

Nach diesem Block ist der Mehrwert weiterer reiner Overlay-Arbeit deutlich kleiner als der Mehrwert eines echten Einspielens mit Build- und Routingprüfung gegen das reale Repo.
